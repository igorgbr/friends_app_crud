
# FriendsApp

**A simple CRUD project through the course  ["Elixir - Programação Funcional"](https://www.udemy.com/gift/programacao-funcional-com-elixir/) by [Jackson Pires](https://www.linkedin.com/in/jackson-pires/).**

## Installation


```shell
asdf install
mix deps.get
```

To create a Fake List:
```shell
mix utils.add_fake_friends
```

To Start:
```shell
mix start
```
